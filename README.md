# Light the World

A interactive world map application that allows you to mark countries you've visited and track your global exploration progress.

## Tech Stack

- React 18
- TypeScript
- Vite
- react-simple-maps
- lucide-react (icons)
- Less (CSS Modules)

## Getting Started

### Prerequisites

- Node.js 16+
- pnpm

### Installation

1. Clone the repository
2. Install dependencies

```bash
pnpm install
```

### Development

Start the development server:

```bash
pnpm dev
```

The app will be available at http://localhost:5173

### Build

Build the project for production:

```bash
pnpm build
```
